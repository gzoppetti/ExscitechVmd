This plugin reads GAMESS output files.

Notes:

Structural Data
The plugin can parse GAMESS output (*.log) files and extract structural information.
Currently we support output files generated by the following GAMESS runtypes:

RUNTYP = ENERGY (extract single point geometry)
RUNTYP = OPTIMIZE (extract trajectory information)
RUNTYP = SADPOINT (extract trajectory information)

We plan to support additional runtypes in future releases.

Electron Orbital Data
Presently, we do not support the ability to display electron orbital data.
This feature is currently under development and will be
present in a rudimentary form in Version 0.2 of the plugin.
